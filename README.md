# Genius Weather App

## 📌 Description
The **Genius Weather App** is a modern and responsive web application built using **React.js** that provides real-time weather information for any city worldwide. It fetches live data from the OpenWeatherMap API and displays temperature, humidity, wind speed, and weather conditions in a clean UI.

---

## 🚀 Features
- 🌍 Search weather by city name  
- 🌡️ Real-time temperature updates  
- 💨 Wind speed and humidity display  
- 🎨 Dynamic UI based on weather conditions  
- 📱 Fully responsive design  

---

## 🛠️ Technologies Used
- React.js  
- JavaScript  
- Tailwind CSS  
- OpenWeatherMap API
